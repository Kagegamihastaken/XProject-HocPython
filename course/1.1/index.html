1.1 - Giới thiệu Python và cách cài đặt Python



A. Giới thiệu về ngôn ngữ Python

    Python là một ngôn ngữ lập trình bậc cao cho các mục đích lập trình đa năng, được phát triển bởi Guido van Rossum và lần đầu ra mắt vào năm 1991. Python được thiết kế các cú pháp dễ đọc hơn các ngôn ngữ lập trình khác và đây là ngôn ngữ được khuyến khích cho những người mới bước vào con đường lập trình. Cấu trúc của Python còn cho phép sử dụng viết mã lệnh với số lần gõ phím tối thiểu. Cho đến 13/8/2023, phiên bản của Python là 3.11.4.
    Python là phần mềm mã nguồn mở, nó có nghĩa là các phần mềm, chương trình, thư viện, ứng dụng được viết trong Python thì sẽ được công khai và sử dụng miễn phí. Bất kì lập trình viên nào cũng có thể tải, chỉnh sửa, tùy biến và đóng góp những thay đổi đó của mình vào cộng đồng chung. Điều này làm giúp cho Python có số lượng thư viện, phần mở rộng khổng lồ được đóng góp từ khắp nới trên thế giới, đáp ứng mọi nhu cầu của lập trình viên, giúp cho cộng đồng Python ngày càng đông đảo, đóng vai trò quan trọng cho truyền đạt kiến thức và giải quyết các vân đề cho mọi người.
    Python hỗ trợ tạo kiểu dữ liệu động và sử dụng cấp phát bộ nhớ tự động, do đó độ phức tạp của Python dược giảm đi đáng kể, người sử dụng không cần phải lo các vấn đề vè khai báo kiểu dữ liệu cho biến, quản lý bộ nhớ, dọn dẹp dữ liệu.
    Python không những có cú pháp đơn giản mà còn không kém mạnh mẽ. Trong thực tế Python có ứng dụng nổi bật ở trong các mảng:
  - Lập trình web
  - Khoa học dữ liệu
  - Trí tuệ nhân tạo (được gọi phổ biến là AI)
    Ngoài ra, Python cũng có thể được sử dụng trong lập trình game, lập trình ứng dụng, viết công cụ tự động hóa (automation tool),... những ứng dụng này không phải là thế mạnh của Python nhưng giúp chúng ta thấy được khả năng đa dạng của ngôn ngữ lập trình này.
    Python cũng đi theo nền tảng khác như Windows, Mac OS, Linux và các hệ điều hành khác thuộc họ Unix.


B. Cách cài đặt Python

    Bạn lên trang web www.python.org rồi bạn nhấn nút "Download". Sau khi bạn nhấn xong nhấn nút "Download Python 3.11.4". Sau khi tải xong bạn mở file bạn vừa tải lên:
    Bạn nhấn nút có chữ "Install Now", khi bạn có hiển thị User Account Control thì bạn nhấn "Yes", mà ngồi chờ cho Python được cài trên máy.
    Nếu bạn đã cài xong, thì mời bạn đọc tiếp phần tiếp theo.
