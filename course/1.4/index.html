1.4 - Chương trình đầu của bạn


    Chúng ta đã học qua cách cài IDE, Python Shell, nhưng làm thế nào bây giờ chúng ta chạy được file Python?


A. Cách tạo file Python bằng phương pháp File Explorer

     + Bước 1: bạn vào "File Explorer"
    Hay là bạn có thể vào "This PC", một số bạn sẽ có This PC ngay ở trong màn hình desktop
.
    Nếu máy của bạn không có "This PC" trong màn hình Desktop, bạn lên mạng xem cách để hiện "This PC" trên màn hình desktop
.      + Bước 2: Sau khi màn hình "File Explorer" xuất hiện như hình dưới:
    Bạn vào tệp mà bạn muốn tạo một chương trình python.
     + Bước 3: Nếu bạn đã tắt "Hide extensions for known file types" ở trong "Folder Option" thì bạn bỏ qua bước này. Trước khi bạn tạo tệp Python, trước tiên vẫn ở trong cửa số "File Explorer", bạn chọn tab "View",
    Sau đó bạn nhấn vào ô tròn được khoang tròn màu đỏ. Lúc này sẽ có cửa số mang tên "Folder Options"
    Sau đó, bạn vào tab "View" ở trong cửa số "Folder Options".
    Tiếp theo đó bạ nhấn vào chỗ được khoanh tròn trong cửa số "Folder Options" sao cho không còn tick nữa, lúc đấy bạn nhấn "OK" hoặc "Apply" rồi "OK"
     + Bước 4: Bạn tìm thấy tệp mà bạn muốn tạo, bạn nhấn chuột phải vào cửa số "File Explorer" và làm như hình dưới.
    Lúc bạn nhấn vào "Text Document" thì tên file nó sẽ phải như "New Text Document.txt", nếu bạn làm có đuôi ".txt" đằng sau, bạn thay đổi tên đuôi từ ".txt" thành ".py"

B. Chạy chương trình đầu của bạn bằng Command Prompt

    Ở trong bài học, 1.3, chúng ta đã được học cách mở "Command Prompt", Đầu tiên bạn mở "Command Prompt". Sau đó bạn gõ câu lệnh sau:
py Địa/chỉ/đến/file/python/của/bạn/<tên file>.py
    Ở trong tệp Python của tôi có dòng lệnh print("Hello World") nếu tôi dùng câu lệnh chạy trên, thì tệp Python của tôi sẽ được chạy và in ra màn hình "Hello World"

C. Cách tạo file Python ở trong Visual Studio Code

    Bạn thấy phương pháp tạo file Python ở trong "File Explorer" rất phức tạp, nhưng ở trong IDE như Visual Studio Code. Nó dược hỗ trợ ngay trong phần mềm luôn.
    Đầu tiên bạn nhấn biểu tương như hình dưới
    Sau khi bạn nhấn xong, nó sẽ hiển thị một tên file và chờ bạn nhập cho file.
    Bạn nhập đuôi nào thì ngôn ngữ lập trình (nếu bạn cài mở rộng của ngôn ngữ lập trình đấy) cũng sẽ thay đổi theo đuôi đấy, ví dụ: ".cpp" định dang thành C++, ".py" định dạng thành Python,...

D. Cách chạy file Python ở trong Visual Studio Code

    Bạn có thể sử dụng phím tổ hợp Ctrl+F5 để chạy một chương trình đang mở trong Visual Studio Code hoặc là bạn nhấn vào tab "Run", sau đó chọn "Run Without Debugging (Ctrl+F5)" thì nó sẽ chạy.

E. Cách tạo file Python ở trong Pycharm

    Pycharm cũng cho tạo file Python, ở trong cửa sổ công cụ "Project".
     + Bước 1: Nhấn chuột phải vào tệp của gốc dự án, chọn File | New...
     + Bước 2: Sau đó bạn chọn "Python File" và bạn sẽ đặt tên cho Python file đấy
    Pycharm sẽ tự động mở file đấy lên nếu bạn đặt tên xong.

F. Cách chạy file Python ở trong Pycharm

    Bạn sử dụng phím tổ hợp Ctrl+Shift+F10 là bạn chạy tệp Python.
