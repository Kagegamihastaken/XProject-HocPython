2.3 - Nhập dữ liệu


    Ở trong bài học này chúng ta sẽ nói về nhập dữ liệu

A. Nhập dữ liệu ở trong console

    Hàm input được dùng để nhập dữ liệu trong console khi đang chạy. Ví dụ, ta muốn tạo một biến dùng để nhập dữ liệu thì ta làm như sau:
bien_nhap = input("Mời bạn nhập: ")


    Ở trên chúng ta khai báo biến bien_nhap rồi gán hàm input vào trong biến bien_nhap, lúc này hàm input in ra "Mời bạn nhập" rồi chờ chúng ta cho dến khi chúng ta nhập dữ liệu.
Mời bạn nhập:


    Sau khi bạn nhập xong, dữ liệu bạn vừa nhập được chuyển thành xâu (đây là mặc định của input). Nếu bạn muốn chuyển sang một kiểu dữ liệu khác thì bạn chỉ cần lấy tên viết tắt của chúng rồi mở ngoặc đóng ngoặc như câu lệnh dưới:
bien_nhap = int(input("Mời bạn nhập: "))


    Lúc này khi bạn nhập số thì thay vì là xầu thì hàm int() chuyển thành kiểu dữ liệu int, nhưng một lưu ý, chỉ hoạt dộng khi nhập dữ liệu kiểu số, nếu ta nhập một kiểu dữ liệu xâu là "test"
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: invalid literal for int() with base 10: 'test'


    Bạn không cần phải dùng hàm int(), mà bạn có thể thay hàm khác như: float(), str(), float(),...
    Đối với số nguyên ta sẽ dùng như sau:
bien_nhap = int(input("Mời bạn nhập: "))


    Đối với số thực ta sẽ dùng như sau:
bien_nhap = float(input("Mời bạn nhập: "))


    Vì bình thường hàm input luôn đưa ra kiểu dữ liệu xâu, nếu bạn muốn nhập kiểu dữ liệu sau, ta sẽ dùng như sau:
bien_nhap = input("Mời bạn nhập: ")
#hoặc
bien_nhap = str(input("Mời bạn nhập: "))


    Lưu ý: bạn không nên để xâu trống trong hàm input
bien_nhap = input("")


    Mặc dù hàm input vẫn cho bạn nhập nhưng dễ bị nhầm lẫn với chương trình bị đứng đơ

B. Cú pháp của hàm input

<tên biến> = input(<kiểu dữ liệu xâu>)


    Trong đó:
     - <tên biến> là tên của biến
     - <kiểu dữ liệu xâu> là xâu bạn in ra rồi chờ người dùng nhập. Nếu bạn nhập kiểu dữ liệu khác xâu thì hàm input sẽ chuyển thành kiểu dữ liệu xâu


Nếu bạn đọc hết chương 2 thì chúc mừng bạn, bạn đã học được xuất dữ liệu, biến và nhập dữ liệu
