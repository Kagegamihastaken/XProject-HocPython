0.2 - Cài đặt IDE để lập trình Python


    IDE là một phần mềm có công cụ giúp đỡ lập trình viên trong việc hát triển phần mềm. Một IDE thường sẽ có một trình soạn thảo mã nguồn, một trình biên dịch hoặc mộ trương trình thông dịch, một công cụ xây dựng tự động, một trình gỡ lỗi và quan trọng hơn nữa là có đánh dấu cú pháp (syntax highlighting).     Có thể bạn nghe bạn có ý định cài, nhưng mà IDE nào? Rất nhiều IDE là miễn phí và bạn có thể cài nhiều IDE trong máy của bạn nếu bạn muốn, vì thế không cần cảm nghĩ là lựa chọn sai IDE.
    Dưới đây là một số IDE chúng tôi khuyên bạn cài.

Visual Studio Code

    Nếu bạn muốn một IDE mà nhẹ hơn Visual Studio và miễn phí thì đây sẽ là lựa chọn của bạn. Visual Studio Code được lập trình bởi Microsoft và hỗ trợ nhiều ngôn ngữ lập trình khác và ba hệ điều hành như: Windows, Linux và Mac OS.
    Các bước để tiến hành cài Visual Studio Code, Đầu tiên bạn lên trang code.visualstudio.com
    Ở đây tôi đang dùng hệ điều hành Windows cho nên cái nút được khoanh tròn màu đỏ nó sẽ thành là "Download for Windows", nếu bạn sử dụng hệ điều hành khác thì nó sẽ hiển thị theo hệ điều hành bạn đang dùng. Sau đó bạn nhấn nút được khoanh tròn và tải về trong máy.
    Tiếp đến bạn mở file setup bạn vừa tải về.


Cách cài đặt Python

    Bạn lên trang web www.python.org rồi bạn nhấn nút "Download". Sau khi bạn nhấn xong nhấn nút "Download Python 3.11.4". Sau khi tải xong bạn mở file bạn vừa tải lên:
    Bạn nhấn nút có chữ "Install Now", khi bạn có hiển thị User Account Control thì bạn nhấn "Yes", mà ngồi chờ cho Python được cài trên máy.
    Nếu bạn đã cài xong, thì mời bạn đọc tiếp phần tiếp theo.
